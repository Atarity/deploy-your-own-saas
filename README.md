- [Pterodactyl](https://github.com/pterodactyl/panel) [⭐️ 8684, 🟢 26d] — game server management panel. It runs game servers in isolated containers while exposing a beautiful web UI to the users.
- [Yacht](https://github.com/SelfhostedPro/Yacht) [⭐️ 47, 🟢 24d] — a web interface for managing docker containers with an emphasis on templating to provide 1 click deployments.
